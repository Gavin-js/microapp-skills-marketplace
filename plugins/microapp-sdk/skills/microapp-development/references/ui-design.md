## 质量基线

始终按高质量营销页面输出，不要生成廉价模板。

### 布局

- 使用 Flexbox 或 Grid 做主布局。
- 维持清晰留白；按 4px/8px 递增节奏排版。
- 让主要区块有明确层级，不要满屏挤压元素。

### 视觉层级

- 使用多层阴影和半透明面板建立深度。
- 重要操作按钮和统计卡片应有更高视觉权重。
- 避免纯白背景上堆普通灰边框卡片的默认模板感。

### 图标纪律

- 每个页面或模块控制在 3 到 5 个图标以内。
- 优先用文字表达功能，装饰元素尽量用 CSS 形状、渐变和光效。
- 保持图标风格统一，不要同时混用多套风格。

### 排版

- 标题和正文要拉开字重对比。
- 正文字号和行高保持易读，不要过度压缩。
- 大屏标题允许更强视觉张力，但不要牺牲信息清晰度。

## 交互和性能约束

- 位移和缩放动画只动 `transform`。
- 淡入淡出只动 `opacity`。
- 高频动画元素使用 `will-change: transform` 或 `translateZ(0)`。
- 移动端对粒子、毛玻璃和大面积模糊做降级。
- 提供 `prefers-reduced-motion` 降级。

优先动画场景：

- 最新签到用户高亮
- 抽奖结果揭晓
- 按钮 hover / active
- 面板切换和模态弹层

避免：

- 用 `top`、`left`、`width`、`height` 做动画
- 在大量动画节点上叠加重 `box-shadow` 和 `filter`
- 同屏堆过多粒子或装饰层

## 风格选择

### `cyber`

适用：

- 科技感活动
- 黑底大屏
- 抽奖、数据展示、舞台感页面

视觉提示：

- 深色背景
- 霓虹青、品红、亮黄作强调色
- 网格、扫描线、发光边框

### `ink`

适用：

- 传统文化、品牌沉稳活动
- 高级问卷或签到页

视觉提示：

- 米色宣纸背景
- 黑、朱红、金色点缀
- 水墨晕染、印章或细线分割

### `crystal`

适用：

- 商务、发布会、简洁高级感页面

视觉提示：

- 亮底或浅色背景
- 玻璃态面板
- 棱镜渐变和柔和高光

### `celebration`

适用：

- 庆典、抽奖、年会、节日场景

视觉提示：

- 暖色系
- 金色、橙红、彩纸元素
- 圆润按钮和更强节奏感

## 风格落地规则

- 先定一个主方向，再让按钮、卡片、标题和装饰保持一致。
- 不要把四种风格混合在同一页面。
- 双端场景允许同风格下做不同密度：移动端更聚焦，PC 端更具舞台感。
